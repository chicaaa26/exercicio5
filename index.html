A linguagem de marcação HTML serve para estruturar e descrever os conteúdos para que os browsers interpretem e apresentem a informação de forma adequada. É composta por tags (elementos) que compõem documentos (ficheiros de texto no formato .html). E que formam Web Sites através das respetivas páginas interligadas acessíveis na Web. Cada página (documento HTML) contém markup estrutural e markup semântico. O Exercício 1 (EX1), pretende desenvolver as fundações de estruturação e marcação de informação no respetivo formato para a Web

Uma página HTML é vulgarmente composta por elementos e conteúdo estrutural e semântico. Normalmente o texto (p. ex. H1, A, ou P) definem a maioria dos elementos, mas o uso de media — imagens bitmap, gráficos vetoriais, vídeo, áudio ou incorporações de elementos interativos como mapas — constituem elementos de conteúdo e de interação importantes.

CSS – Cascading Style Sheets – é uma linguagem de estilos para descrever a apresentação (design visual e interação, ou animações limitadas/predeterminadas) de um documento estruturado em HTML (ou XML, incluindo SVG). De uma forma simples, vamos “fazer bonito:”

O Box Model é uma representação de “todos” os elementos (sejam eles contentores de conteúdo de natureza estrutural ou semântica) de HTML. Cada elemento HTML é constituído por um conjunto de propriedades que podem ser manipuladas de forma global, ou cada uma independentemente.
